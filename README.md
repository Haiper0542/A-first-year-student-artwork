# High Tech
* #### 장르 : Fps게임
* #### 역할 : 개인 습작
* #### 개발 언어, 툴 : C#, Unity

## 프로젝트 소개
### 게임 소개
* Fps장르를 연습하기 위해 제작된 게임입니다.
* F키를 눌러 아이템을 주울 수 있으며, 휠키로 아이템을 던질 수 있습니다.
* 좌클릭으로 총을 발사할 수 있으며, R키를 통해 재장전이 가능합니다.
* 총에는 탄퍼짐이 존재하며 3발이상 발사할 시 점차 집탄률이 낮아지게 됩니다.
+ **아이템**
  + 총 : 좌클릭을 통해 총알을 발사하며 우클릭으로 줌을 할 수 있습니다.
  + 총알 : 총을 들고 R키를 누를 시 재장전이 가능하며, 해당 아이템은 재장전할 때마다 소모됩니다.
  + 수류탄 : 우클릭을 통해 준비가 가능하며, 3초후 폭발하게 됩니다. 휠키를 통해 멀리 던질 수 있습니다.

# Runner
* #### 장르 : 런게임
* #### 역할 : 프로그래밍
* #### 개발 언어, 툴 : C#, Unity
* #### 사용 기기 : 마이크로 비트

## 프로젝트 소개
### 게임 소개
* Maker Heroes에 참가하기 위해 제작된 게임입니다.
* 앞으로 달려나가면서 장애물들을 피하는 게임입니다.
* 마이크로 비트의 센서를 이용하여 실제 플레이어가 점프하거나 숙이거나 옆으로 피하는 것으로 조작이 가능합니다.
* 코인을 먹음으로써 점수를 얻을 수 있습니다.
* 모든 장애물은 데미지를 1씩 주게 되며, 5번 맞을 시 게임 오버됩니다.
+ **아이템**
  + 전기 철조망, 차량, 파이프, 가속 방지턱 : 점프로만 회피가능합니다.
  + 미사일 : 화면에 느낌표 표시가 뜨게 되며, 숙이기로 회피할 경우 계속 나아가며 다른 장애물을 파괴합니다.
